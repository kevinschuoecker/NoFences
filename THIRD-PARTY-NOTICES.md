## Shell context menu implementation

`FlowGrid/Win32/ShellContextMenu.cs` is based on "FileBrowser" by Andreas
Johansson, originally published on CodeProject
(http://www.codeproject.com/useritems/FileBrowser.asp), with hooking code
from the MSDN Magazine Cutting Edge column. This component is scheduled to
be replaced by an in-house implementation.
